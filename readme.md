# Can Label Viewer
Dette er et simpelt projekt, der viser, hvordan man kan vise og ændre labels på 3D-dåser i Three.js.

## Installation
1. Clone dette repository.
2. Kør `npm install` for at installere afhængigheder.
3. Kør `npm run dev` for at starte udviklingsserveren.
4. Åbn `http://localhost:5173` i din browser for at se projektet.

## Funktioner
- Visning af 3D-dåser med labels.
- JSON fil med dynamisk opdatering af label-størrelse og -position.
- Thumbnails for at vælge forskellige dåser.
